# Discord 英語內容編輯器

輕鬆管理每日英語學習內容的網頁編輯器。

## 功能說明

### 檔案列表
- 顯示 Day 01-31 的檔案狀態
- 綠色：已在 GitHub
- 黃色：僅本地暫存
- 灰色：尚未建立

### 基本操作
- **載入**：從 GitHub 或本地暫存載入內容
- **儲存**：儲存到瀏覽器本地暫存 (localStorage)
- **下載**：下載 JSON 檔案到電腦
- **清空**：清除編輯器內容

### 上傳 JSON 檔案
選取日期後，可點擊「上傳 JSON 檔案」按鈕匯入現有的 JSON 檔案：
1. 先從下拉選單選擇日期 (Day 01-31)
2. 點擊「📤 上傳 JSON 檔案」按鈕
3. 選擇要匯入的 `.json` 檔案
4. 檔案內容會自動填入編輯器並儲存到本地暫存

**JSON 檔案格式：**
```json
{
  "content": "訊息內容..."
}
```

## 檔案結構

```
├── index.html          # 主編輯器頁面
└── contents/           # 每日內容 JSON 檔案
    ├── day01.json
    ├── day02.json
    └── ...
```
